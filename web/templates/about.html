About EV Opinion Search Engine

Project Overview

The EV Opinion Search Engine is a specialized search platform designed to help users discover, analyze, and understand public opinions about electric vehicles. By gathering and processing user-generated content from platforms like Reddit, our system provides valuable insights into consumer sentiment, trending topics, and common concerns about EVs.

Key Features

Advanced Search

Powerful search functionality with filtering by sentiment, date, source, and more.

Sentiment Analysis

Automatic classification of opinions as positive, negative, or neutral.

Topic Modeling

Identification of key topics and themes across EV discussions.

Visual Analytics

Interactive charts and visualizations to understand opinion trends.

Technical Implementation

This project is built using a modern technology stack:

  • Data Collection: Python-based crawler using PRAW for Reddit API access
  • Indexing: Apache Solr for fast and scalable search functionality
  • Classification: Transformer-based sentiment analysis and LDA topic modeling
  • Web Interface: Flask web framework with Bootstrap UI

Dataset

Our search engine is built on a dataset of over 10,000 opinions about electric vehicles collected from multiple Reddit communities. The dataset includes posts and comments from r/electricvehicles, r/teslamotors, r/cars, and other relevant subreddits.

Dataset Statistics:
  • Total records: ~12,000
  • Time period: 2021-2023
  • Unique authors: ~5,000
  • Average text length: 105 words

Use Cases

The EV Opinion Search Engine can be valuable for:

  • Market Researchers: Understanding consumer perception of EV brands and models
  • Product Developers: Identifying common concerns and feature requests
  • Potential Buyers: Researching real-world experiences with specific EVs
  • Policy Makers: Gauging public opinion on EV infrastructure and incentives
